## Customization

### Update Personal Information

1. **Name and Title**: Edit the hero section in `index.html`
2. **About Section**: Modify the about text in the About section
3. **Skills**: Update skills in the Skills section
4. **Projects**: Replace project cards with your actual projects
5. **Contact Info**: Update contact details in the Contact section

### Change Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... */
}
```

### Modify Typing Text

Edit the typing text in `index.html`:

```html
<span class="typing-text">Web Developer</span>
```

You can also modify the animation timing in `styles.css` under the `.typing-text` class.
